# Nmap

## Overview

Nmap (Network Mapper) is one of the most widely used network reconnaissance and security auditing tools.

It can be used to:

- Discover hosts on a network
- Identify open ports
- Detect running services
- Determine service versions
- Perform operating system fingerprinting
- Execute NSE (Nmap Scripting Engine) scripts

## Installation

Kali Linux usually includes Nmap by default.

Verify installation:

bash nmap --version 

Install if necessary:

bash sudo apt update sudo apt install nmap 

## Basic Workflow

1. Discover live hosts
2. Identify open ports
3. Detect services
4. Determine versions
5. Perform additional enumeration

## Common Use Cases

- Network inventory
- Security assessments
- Service enumeration
- Vulnerability discovery
- Lab exercises
