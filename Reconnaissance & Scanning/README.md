# Reconnaissance & Scanning

This repository contains notes, commands, examples, and documentation for reconnaissance and scanning tools commonly used in cybersecurity assessments and penetration testing.

## Objectives

- Learn network discovery techniques
- Understand service enumeration
- Perform DNS and domain reconnaissance
- Gather publicly available information (OSINT)
- Build a reusable knowledge base for future assessments

## Tools Covered

### Nmap
Network scanning and service enumeration.

### Netdiscover
ARP-based host discovery on local networks.

### Whois
Domain registration and ownership lookup.

### Dig
DNS interrogation and record enumeration.

### theHarvester
OSINT tool for collecting emails, subdomains, hosts, and metadata from public sources.

## Repository Structure

text Reconnaissance-Scanning/ ├── nmap/ ├── netdiscover/ ├── whois/ ├── dig/ ├── theharvester/ ├── scripts/ └── notes/ 

## Disclaimer

Only perform reconnaissance and scanning activities on systems and networks you own or have explicit authorization to assess.

Unauthorized scanning may violate laws, regulations, or organizational policies.
