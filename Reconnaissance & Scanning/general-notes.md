# General Notes

## Reconnaissance Overview

Reconnaissance is the process of gathering information about a target before attempting deeper enumeration or security testing.

### Types of Reconnaissance

#### Passive Reconnaissance

Information gathering without directly interacting with the target.

Examples:

- Whois lookups
- Search engines
- DNS records
- Public breach data
- Social media research

#### Active Reconnaissance

Information gathering that directly interacts with target systems.

Examples:

- Nmap scans
- Port scanning
- Service enumeration
- Network discovery

## Typical Workflow

1. Identify targets
2. Discover live hosts
3. Enumerate open ports
4. Identify services
5. Gather DNS information
6. Collect OSINT
7. Document findings

## Documentation Tips

For every assessment:

- Record commands used
- Save scan results
- Document observations
- Note unusual services
- Keep screenshots when relevant

## Common Questions

### What is a port?

A logical communication endpoint used by services.

### What is a service?

Software listening on a port.

### What is enumeration?

The process of extracting detailed information from discovered services.

### What is OSINT?

Open Source Intelligence gathered from publicly available information.
