# theHarvester

## Overview

theHarvester is an Open Source Intelligence (OSINT) tool used to gather publicly available information about a target domain.

It can collect:

- Email addresses
- Subdomains
- Hostnames
- IP addresses
- Employee information (depending on source)
- Publicly available metadata

theHarvester is commonly used during passive reconnaissance to build a profile of an organization's Internet-facing infrastructure.

## Why Use theHarvester?

theHarvester helps answer questions such as:

- What subdomains exist?
- What email addresses are publicly exposed?
- What hosts are associated with the organization?
- What infrastructure can be identified through public sources?

## Information Sources

theHarvester can query sources such as:

- Bing
- DuckDuckGo
- Brave
- SecurityTrails
- Shodan
- Censys
- Hunter.io
- Intelligence X
- VirusTotal

Some sources require API keys.

## Typical Workflow

```text
Identify Target Domain
         ↓
Select Data Source
         ↓
Run theHarvester
         ↓
Collect Results
         ↓
Validate Findings
         ↓
Document Findings
```
