# Wireshark

## Overview

Wireshark is the world's most widely used network protocol analyzer. It allows users to capture, inspect, and analyze network traffic in real time or from previously saved packet capture (PCAP) files.

Wireshark is used by:

- Security Analysts
- SOC Analysts
- Incident Responders
- Network Engineers
- Penetration Testers
- Malware Analysts

---

## What Can Wireshark Do?

- Capture live network traffic
- Open and analyze PCAP files
- Decode hundreds of network protocols
- Follow TCP and UDP streams
- Inspect packet contents
- Troubleshoot network issues
- Investigate security incidents

---

## Why Learn Wireshark?

Wireshark helps you understand:

- How devices communicate
- TCP/IP networking
- DNS resolution
- Web traffic
- Encryption
- Network troubleshooting
- Security investigations

---

## Key Features

### Packet Capture

Capture packets from network interfaces.

### Display Filters

Filter packets to focus on specific traffic.

### Packet Inspection

View detailed protocol information.

### Statistics

Analyze conversations, endpoints, and protocol usage.

### Exporting

Save selected packets or complete captures.
