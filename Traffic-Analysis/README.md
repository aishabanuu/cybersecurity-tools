# Traffic Analysis

## Overview

This repository contains notes, commands, examples, packet captures, and protocol documentation for learning network traffic analysis using Wireshark and tcpdump.

Traffic analysis is a fundamental cybersecurity skill used by SOC analysts, incident responders, malware analysts, network administrators, and penetration testers to understand how devices communicate across a network.

## Objectives

- Understand packet capture fundamentals
- Learn common networking protocols
- Analyze network traffic
- Practice using Wireshark and tcpdump
- Interpret packet headers and payloads
- Build practical packet analysis skills

## Tools Covered

### Wireshark

A graphical packet analyzer used to capture, inspect, and troubleshoot network traffic.

Topics include:

- Packet capture
- Display filters
- Protocol analysis
- TCP Streams
- Statistics
- Packet inspection

---

### tcpdump

A lightweight command-line packet capture tool.

Topics include:

- Live packet capture
- Packet filtering
- Saving PCAP files
- Reading captures
- Network troubleshooting
