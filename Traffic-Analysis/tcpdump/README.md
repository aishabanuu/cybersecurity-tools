# tcpdump

## Overview

tcpdump is a command-line packet analyzer used to capture, inspect, and troubleshoot network traffic. It is lightweight, fast, and commonly available on Linux and Unix systems.

Unlike Wireshark, tcpdump operates entirely from the terminal, making it ideal for remote servers, cloud environments, and systems without a graphical interface.

## Why Learn tcpdump?

tcpdump helps security professionals:

- Capture live network traffic
- Troubleshoot connectivity issues
- Analyze protocols
- Investigate security incidents
- Create PCAP files for later analysis in Wireshark

## Features

- Live packet capture
- Packet filtering
- Protocol inspection
- Save captures to PCAP
- Read existing PCAP files
- Capture on specific interfaces

## Learning Objectives

- Learn packet capture from the command line
- Understand Berkeley Packet Filters (BPF)
- Capture traffic for later analysis
- Troubleshoot network communication
- Analyze common network protocols

## Typical Workflow

```text
Select Interface
       ↓
Apply Capture Filter
       ↓
Capture Traffic
       ↓
Save PCAP
       ↓
Analyze with Wireshark
```
