# SQLMap

## Overview

SQLMap is an open-source tool used to assess web applications for SQL injection vulnerabilities and to understand how applications interact with databases.

For beginners, SQLMap should be used in controlled lab environments to learn how database-backed applications process user input and how security weaknesses can arise when input is not handled safely.

## What SQLMap Does

- Tests parameters for SQL injection vulnerabilities
- Identifies backend database technologies
- Analyzes application inputs
- Generates detailed assessment results
- Helps validate findings during authorized security assessments

## Supported Databases

- MySQL
- PostgreSQL
- Microsoft SQL Server
- Oracle
- SQLite
- MariaDB

## Learning Objectives

- Understand SQL injection concepts
- Learn how web applications interact with databases
- Practice identifying input parameters
- Analyze assessment results
- Improve web application security knowledge

## Recommended Learning Environment

Practice only in safe, intentionally vulnerable labs such as:

- DVWA
- OWASP Juice Shop
- WebGoat
- Metasploitable

## Typical Workflow

```text
Identify Input Parameters
          ↓
Capture Requests
          ↓
Analyze Parameters
          ↓
Run Assessment
          ↓
Review Results
          ↓
Document Findings
```

## Important Note

Always ensure you have authorization before testing a web application. Unauthorized security testing may violate laws or organizational policies.
