# Burp Suite

## Overview

Burp Suite is one of the most popular web application security testing tools.

It acts as a proxy between your browser and a web application, allowing you to inspect, modify, and analyze HTTP and HTTPS traffic.

Burp Suite is commonly used during:

- Web application assessments
- Bug bounty research
- Security audits
- API testing
- Application mapping

## Main Components

### Proxy

Intercept and inspect requests and responses.

### Target

Map website structure and discovered content.

### Repeater

Replay and modify requests manually.

### Intruder

Automate testing of parameters and inputs.

### Decoder

Encode and decode data.

### Comparer

Compare requests and responses.

## Why Learn Burp Suite?

Understanding Burp helps you:

- Learn HTTP fundamentals
- Understand web application behavior
- Analyze authentication mechanisms
- Identify security weaknesses
- Document application functionality

## Typical Workflow

```text
Browser
   ↓
Burp Proxy
   ↓
Web Application
```
