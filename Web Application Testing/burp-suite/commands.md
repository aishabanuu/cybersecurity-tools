# Burp Suite Commands and Features

## Launch Burp

```bash
burpsuite
```

---

## Open Browser

```text
Proxy → Intercept → Open Browser
```

---

## Enable Intercept

```text
Proxy → Intercept
```

---

## Send Request to Repeater

Right Click Request

```text
Send to Repeater
```

Shortcut:

```text
Ctrl + R
```

---

## Send Request to Intruder

Right Click Request

```text
Send to Intruder
```

Shortcut:

```text
Ctrl + I
```

---

## View Target Structure

```text
Target → Site Map
```

---

## View HTTP History

```text
Proxy → HTTP History
```

---

## Decode Data

```text
Decoder
```

Useful for:

- URL encoding
- Base64
- Hex

---

## Compare Responses

```text
Comparer
```

Useful for analyzing differences between requests and responses.
